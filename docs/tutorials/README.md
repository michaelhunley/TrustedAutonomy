- [Blender Plugin Walkthrough](./blender-plugin-walkthrough.md) — Create a Blender Python addon using TA's `blender-addon` template and guided onboarding.

## Getting Started

If you are new to TA, see [USAGE.md](../USAGE.md) for the main documentation.

The quickest way to start a new project:

```bash
# Install a template
ta template install blender-addon

# Create a new project
ta new run --template blender-addon --name my-addon

# Start the first goal
ta run "Scaffold addon structure"
```
